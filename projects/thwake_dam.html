Thwake Dam Catchment Model

This project focused on delineating the Thwake Dam Catchment area and mapping key environmental features to support environmental planning and impact assessment.

Methodology

  1. Collected base spatial data including DEMs
  2. Delineated the Thwake Dam Catchment using hydrological tools in QGIS
  3. Extracted and mapped river networks within the catchment
  4. Identified and mapped dumpsites along the river network

Tools Used

  • QGIS
  • Google Earth Pro
  • Survey123

Outputs

  • Catchment boundary map
  • River network
  • Dumpsites distribution map
