Real-Time Incident Reporting Mobile Application - Ongoing

This project involves the development of a civic-tech mobile application designed to enable members of the public to report real-time incidents such as crime, floods, road accidents, and infrastructure hazards. The application was motivated by the need for faster, more transparent, more accessible reporting mechanisms in urban environments such as Nairobi.

In many parts of Kenya, delays in emergency response and limited public access to reliable incident data reduce community safety and preparedness. This application seeks to bridge that gap by providing a simple platform through which citizens can submit geo-referenced incident reports, supporting improved situational awareness for both public and relevant response agencies.

Core Features

  1. User-friendly mobile interface for submitting incident reports.
  2. Icon and Description upload or incident verification
  3. Backend API for storing and managing incident data.
  4. Real-time data synchronization
  5. Basic user authentication and validation

Planned Enhancements

  • Interactive incident map and filtering tools
  • Administrative dashboard for authorities
  • Incident status tracking and updates
  • Push notifications for critical alerts
  • Role-based access control

Technologies

  • Flutter
  • Laravel
  • MySQL
